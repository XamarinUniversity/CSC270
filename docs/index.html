Before you begin

This is a cross-platform class which can use either iOS or Android. To complete the coding exercises, you will need a Xamarin development environment with either an emulator/simulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This class is an overview of how your Xamarin (and .NET) applications manage memory using a Garbage Collector. Most of the information in this class is relevant to either Mono or .NET and the differences between the platforms are pointed out as part of the discussion.

Lecture

CSC270 - Garbage Collection Fundamentals.

Objectives

  1. Explain the Garbage Collector

Demonstrations

There are two demo projects included in the materials so you can experiment and reproduce the demonstrations from the class.
